# qtrade 决策手册 — 系统买什么、何时买卖、为什么

这份文档用大白话回答四个问题。所有规则都能在代码里找到对应（括号内），
且都经过 7 年数据验证（research/log.md E23）。

## 1. 买什么（品种选择）

**当前池子：10 个 OKX 永续合约**（BTC/ETH/SOL/XRP/DOGE/ADA/LTC/LINK/AVAX/DOT）。

为什么是这十个——不是拍脑袋，是三条硬规则（research/universe_score.py）：

1. **边际贡献为正**：把任何一个从组合里拿掉，组合夏普都会下降（当前最弱的 XRP 也贡献 +0.03，最强的 AVAX +0.32）
2. **候选品种边际贡献为负**：测过 UNI/ATOM/FIL/NEAR/TRX/BCH，加进来全都拖累组合（-0.6~-0.8），因为它们和现有池高度相关、只添加噪声
3. **季度重评，不追排名**：换池必须过"双面板预注册门槛"，防止追涨杀跌式的选池

为什么是 crypto 而不是 A股/美股：不是偏好，是**四次实验的结论**（E19-E22）——
免费价格数据 + 技术信号在股票市场挖不出可靠优势，在 crypto 能。

## 2. 什么时候买（入场）

系统有两条独立的入场逻辑，各管一半资金风险：

**趋势腿（CTA）**：三个时间尺度（4天/12天/30天）各投一票，
看快均线是否在慢均线上方。三票平均 = 仓位方向和大小。
- 三票全看多 → 满票做多；两多一空 → 1/3 仓做多；以此类推
- *通俗说：市场持续涨了一段时间，跟着买；持续跌，跟着卖空*

**回归腿（布林 z-score + regime 对齐）**：
- 价格偏离 4 天均值超过 2 个标准差（z ≤ -2 超卖 / z ≥ +2 超买）时反向入场
- **但有 regime 闸门**：价格在月线均线上方只准买超卖（牛市买回调），
  下方只准卖超买（熊市卖反弹）——顺大势、逆小势
- *通俗说：涨势里等急跌抄底，跌势里等反弹做空，绝不逆着大方向接飞刀*

## 3. 什么时候卖（出场与风控）

- **趋势腿**：均线票数变了就变仓——趋势结束的信号就是出场信号
- **回归腿**：价格回到均值（z 过 0）即止盈离场；**没有价格止损**——
  E9 证明对均值回归设价格止损等于在期望收益最大点割肉
- **风险靠仓位控制而非止损**：每个品种按波动率反比定仓
  （波动率 40% 目标 / 实际波动率，上限 1 倍），市场越疯仓位越小
- **组合级熔断**：从高水位回撤 -20% → 全部清仓停机，人工排查后才能重启
  （一年内触发概率约 6%，蒙特卡洛 E29）

## 4. 为什么可以信这套规则

- **七年验证**：参数在 2023-26 调的，在从未见过的 2019-22 上 Sharpe 1.11（E23）
- **五次自我挑战全部失败**：参数集成/组合缩放/基差套利/扩池/换周期，
  没有一个能按预注册标准打败现构造（E24-E28）
- **已知弱点明码标价**：单边暴涨年会跑输甚至小亏（2024：-14%），约 5-7 年一遇；
  预期管理：中位年化 8-14%，约 1/4 年份亏损（E29）
- **每笔决策可审计**：`python -m qtrade.cli explain` 随时展示每个品种的完整决策链
